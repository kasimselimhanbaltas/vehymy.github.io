### God of War Mini Web Game!
This is a project that I developed in the 3rd grade. It's nothing much but just fun :)

[Live(Github Pages)](https://vehymy.github.io/)

### Controls
Movement: ↑←↓→ or WASD  
Axe: Left click  
Blades: Right click  
Special Skill: Q - Q
